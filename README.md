# Mini-Project – Java Mini Projects Collection

## Description
This repository contains beginner-friendly Java mini projects developed for practicing core Java concepts and Object-Oriented Programming (OOP).

The projects are console-based and help in improving problem-solving skills, programming logic, and understanding of Java fundamentals.

---

# Projects Included

## 1. Guessing Game

### Overview
Guessing Game is a simple Java console application where the computer generates a random number and the player tries to guess it correctly.

The program gives hints such as:
- Too High
- Too Low
- Correct Guess

### Features
- Random number generation
- Hint system
- Attempt counter
- Console interaction

### Concepts Used
- Loops
- Conditional Statements
- Random Class
- Scanner Class
- User Input Handling

---

## 2. Student Management System

### Overview
Student Management System is a Java console-based application used to manage student records.

Users can:
- Add students
- View students
- Search student records
- Update student information
- Delete student records

### Features
- CRUD Operations
- Menu-driven system
- Student data handling
- Search functionality

### Concepts Used
- Classes and Objects
- Encapsulation
- Constructors
- ArrayList
- Methods
- Loops and Conditions
